# BracketSystemProject

vejledning:

1) det heller er i php og css filer, brug xampp til at hoste mappen hvor alle filerne ligge i.

2) database kan genoprettes ved at importer vores database i database mappen hvis du går in på phpmyadmin og lave en database med navnet "bracketsystem" og så importere filen.

3) det er forsøgt at så dynamsik som muligt på den tid vi har haft.

mangler/kommende ting:

1) password krypterin

2) Sættet af flere teams en der er plads til skal komme med en fejl melding

3) vælge en vinder af kampen så de kan kommer videre.

4) profil, kamp, om os mangler at bliver sat

5) mangler navn til siden

6) tilpassing til alle skærme

7) side til at oprette turneringer

8) navigations bare skal laves dynamisk
